# 1만시간의 법칙
 피그마 디자인 시안을 제공받아 만든 반응형 웹 페이지입니다.


## 1. 목표와 기능
 - 모바일 페이지와 PC 페이지를 반응형 웹으로 제작한다.
 - 유저에게 하루 공부 시간을 입력 받아 1만 시간이 되려면은 며칠 걸리는지 계산해서 알려준다. (추후 JS 추가 예정)
 - "훈련 하러 가기 GO!GO!" 버튼 클릭시 모달 창이 뜨게 한다. (추후 JS 추가 예정)
